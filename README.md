# Amazon Manual Testing Project

##  Project Overview
This project demonstrates manual functional testing performed on the Amazon web application. It includes test case design, execution, bug reporting, and basic API testing using Postman.

##  Objectives
- Validate core functionalities of Amazon
- Identify functional defects
- Perform API testing using Postman
- Document testing process professionally

- Postman

- Response validation (Status Code: 200 OK)
