##  Project Overview
This project demonstrates manual functional testing performed on the Amazon web application. It includes test case design, execution, bug reporting, and basic API testing using Postman.

## API Testing using Postman

API testing was performed using Postman to validate backend response.

### Test Details:
- Method: GET
- Endpoint: https://fakestoreapi.com/products
- Status Code: 200 OK
Note: Amazon APIs are secured, so a public API was used for demonstrating API testing.
### Result:
The API returned a successful response with product data in JSON format
